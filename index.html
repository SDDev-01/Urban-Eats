Curso de Diseño Web y Experiencia de Usuario

Aprende HTML5, CSS3 y JavaScript creando una aplicación web desde cero.

Ubicación

Contenido del curso

Unidad 1. Fundamentos de Diseño Web y UX

  • Conceptos básicos: cliente, servidor, dominio y hosting.
  • Diferencia entre UI y UX.
  • Usabilidad, accesibilidad y navegación intuitiva.

Unidad 2. Estructura del Contenido con HTML5

  • Estructura semántica con <header>, <nav>, <section>, <article>, <footer>.
  • Formularios, tablas, listas, imágenes y enlaces.

Unidad 3. Diseño Visual y Responsive con CSS3

  • Color, tipografía, espaciado y alineación.
  • Flexbox, Grid y Media Queries.

Unidad 4. Interactividad con JavaScript

  • Variables, funciones y eventos.
  • Manipulación del DOM y validación básica.
